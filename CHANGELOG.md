# Changelog

## v0.1.0 - Initial Public Release

- Added playable Nibbles-inspired snake gameplay.
- Added deterministic levels 1-10 with static obstacles.
- Added Normal, Hard, and Insane difficulty modes.
- Added lives, level-complete, you-died, game-over, and high-score flows.
- Added generated retro sound effects with mute support.
- Added persistent local settings and top-10 high scores per difficulty.
- Added Linux release packaging through `make release-linux`.
