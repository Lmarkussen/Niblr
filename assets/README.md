# Assets

Niblr currently uses procedural graphics, generated sound effects, and generated chiptune-style background music.

No downloaded or copyrighted music/assets are bundled. The menu, early-level, and late-level music loops are generated in code by `synthMusic` in `main.go` using simple square-wave note patterns.

If external assets are added later, document the source URL, author, license, file name, and download date here before committing them.
